Parameters
$array : array<string|int, mixed>

An array.

$filter : Closure

A filter to apply on each entry. If no callback is supplied, all empty entries of array will be removed. See empty() for how PHP defines empty in this case.

$mode : int = 0

Flag determining what arguments are sent to callback:

  • ARRAY_FILTER_USE_KEY - pass key as the only argument to callback instead of the value
  • ARRAY_FILTER_USE_BOTH - pass both value and key as arguments to callback instead of the value Default is 0 which will pass value as the only argument to callback instead.
Return values
array<string|int, array<string|int, mixed>> —

A list of two arrays where $list[0] are the entries validated by the filter and $list[1] are the remaining entries not filtered of the array.
