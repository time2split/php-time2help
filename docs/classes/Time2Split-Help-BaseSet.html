BaseSet extends ArrayAccess, Countable, Traversable
in
  • time2help
  • container

BaseSet.php : 27

A set data-structure to store elements of type T.

An element can only be assigned once in a Baseset. The comparison operation to check if two elements are equals depends on the set implementation.

A Baseset provides the array syntax facilities to query and modify the set contents, however its only facilities: a set must not be considered as an array. A Baseset<T> stores instances of T, there is no key semantic.

This library always provides implementations of BaseSet extending the abstract class Set. The class Sets provides static factory methods to create instances of Set.
